# PHP Quiz Application (20 MCQs)

Members : gohilmitrajsinh

## 📌 Project Overview

This is a simple web-based Quiz Application developed using PHP.  
The application presents 20 multiple-choice questions and calculates the user's score based on their selected answers.

This project demonstrates core PHP concepts such as:
- Arrays
- Loops (foreach)
- Conditional statements (if-else)
- Form handling using POST method
- Server-side processing
- Dynamic result display

---

## 🚀 Features

- 20 Multiple Choice Questions
- Single correct answer system
- Server-side validation
- Automatic score calculation
- Feedback for correct and incorrect answers
- Clean and simple user interface
